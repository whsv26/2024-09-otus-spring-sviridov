## C4 Model of the Application

The C4 model consists of:
- System Context (L1)
- Container (L2)
- Component (L3)
- Code (L4)

The application includes five systems:
- API Gateway (gateway)
- User Management (user)
- Web Novel Content Management (novel)
- Web Novel Rating (rating)
- Web Novel Search (search)
